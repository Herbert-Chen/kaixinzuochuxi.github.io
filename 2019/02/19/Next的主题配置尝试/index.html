Next的主题配置尝试

Veröffentlicht am 2019-02-19

主要内容

本文就主题选择，应用，部分参数配置与碰到的一些问题进行了简单的总结。后续遇到新的问题会继续补充。
主要参考博客为http://saili.science/2017/04/02/github-for-win/

主题的选择

在GITHUB+HEXO搭建博客的第一个链接中给出了一些主题网站，里面可选的有很多，经过一些考虑，最终还是选择了Next主题。

具体原因如下：

  1. 产品免费且受众广
  2. 功能上支持中文，支持文章目录生成。

主题的应用

按照主题网站提示即可。

注意以后版本更新可能会导致命令(–branch v5.1.2)有一些差异，具体以主题网站为主。

此种方式也可用于配置其他主题，即

Automatically Excerpt. Not recommend.

Use in the post to control excerpt accurately.

auto_excerpt:
enable: true
length: 150

Read more button

If true, the read more button would be displayed in excerpt section.

read_more_btn: true
length可以自己设置。
