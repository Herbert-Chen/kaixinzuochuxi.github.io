遇到的问题

安装matlab.engine时python没有管理员权限

这个问题是由于管理员限制matlab文件夹的权限造成的。问题会不会出现取决于当前是否为管理员用户。如果是，则不存在这个问题，如果不是，需要给定管理员权限。

最简单的方法，开始菜单中搜索cmd，右键选择管理员运行，然后cd到目标目录下，再用python安装即可。
